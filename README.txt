DSSC Mental Hub

DSSC Mental Hub is a web-based mental wellness platform designed to
support users with mental health resources, guidance, and interactive
tools. The system provides a clean and responsive interface where users
can access helpful content, track their mental well-being, and interact
with different wellness features.

This project is built using React, TypeScript, Vite, Tailwind CSS, and
Supabase for backend services.

------------------------------------------------------------------------

FEATURES

-   User Authentication (Supabase)
-   Mental wellness tracking
-   Mental health resources
-   Responsive modern UI
-   Light / Dark theme support
-   Fast performance using Vite
-   Cloud database integration with Supabase

------------------------------------------------------------------------

TECH STACK

Frontend - React - TypeScript - Vite - Tailwind CSS - ShadCN UI - Radix
UI

Backend / Database - Supabase

Additional Libraries - React Router - React Hook Form - TanStack React
Query - Zod Validation - Recharts - Lucide Icons

INSTALLATION

1.  Clone the repository

git clone https://github.com/your-username/dssc-mental-hub.git

2.  Navigate to the project

cd dssc-mental-hub

3.  Install dependencies

npm install

4.  Run the development server

npm run dev

The application will start at: http://localhost:5173

------------------------------------------------------------------------

BUILD FOR PRODUCTION

npm run build

Preview production build:

npm run preview

------------------------------------------------------------------------

ENVIRONMENT VARIABLES

Create a .env file in the root directory.

Example:

VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_key
